# EventOps Color Palette

## Core Colors (from palette)
- **Vivid Orange**: `#FF9F1C` - Primary accent, CTAs, buttons
- **Hunyadi Yellow**: `#FFBF69` - Secondary accent, highlights
- **Buff**: `#CB997E` - Warm neutral, borders
- **Pale Peach**: `#FFE8D6` - Light backgrounds, cards

## Derived Colors

### Warm Tones
- **Soft Coral**: `#F4A261` - Notifications, warnings
- **Terracotta**: `#E76F51` - Error states, alerts
- **Maroon/Wine**: `#9C4A3B` - Dark accent, severity

### Cool Tones (complementary)
- **Sage Green**: `#87A878` - Success states, confirmations
- **Teal**: `#5C9EAD` - Info, links
- **Muted Blue**: `#6B8CAE` - Secondary info
- **Soft Purple**: `#9B8AAD` - Tags, categories
- **Dusty Pink**: `#D4A5A5` - Soft accents

### Neutrals
- **Off-Black**: `#1F1F1F` - Primary text, dark backgrounds
- **Charcoal**: `#2D2D2D` - Secondary dark
- **Dark Gray**: `#4A4A4A` - Muted text
- **Medium Gray**: `#7A7A7A` - Placeholders
- **Light Gray**: `#E8E8E8` - Borders, dividers
- **Cream/Off-White**: `#FAF8F5` - Light backgrounds

## Usage Guidelines
- Use Vivid Orange for primary actions (Send, Create, Submit)
- Use Sage Green for success/confirmation
- Use Terracotta for errors/warnings
- Off-Black (#1F1F1F) instead of true black (#000000)
- Pale Peach for card backgrounds in light mode
